
Individual book page
- QR code removed: The original shows a QR code block; the edited version does not. This was changed because the QR-code lead to the same page and was thus unnecessary
- Social share icons: The original uses CSS sprite-based icons. The edited version replaces all of them with inline SVG icons (Mastodon, Twitter/X, Facebook, Bluesky, Print).
- "Share" heading added: The edited version adds a "Share this book with your friends" label above the social buttons. This was done so there is more of a call to action
- "Summary" heading added: The edited version adds an <h2 class="summary-heading">Summary</h2> before the book description. The original has no heading there.
- Download section redesigned: The original uses a full HTML table listing all formats. The edited version replaces this with a prominent "Read Now!" button and a collapsible dropdown for all download options, styled as individual rows instead of table cells. This way we hoped to make it more clear visually
- "Similar Books" section redesigned: The original uses a simple navlink list for bookshelves. The edited version replaces this with a horizontally scrollable book carousel (with cover images for Dracula, Dorian Gray, Jekyll & Hyde, etc.) and a grid of category buttons.
- Section order swapped: In the original, the order inside the page is Download -> Similar Books -> About this eBook. In the edited version it is Download -> About this eBook -> Similar Books.
- Subjects table row changed: The original lists each subject in a separate table row. The edited version puts all subjects into a single table cell separated by bullet characters.
- "Back" button added: The edited version adds a Back button near the top of the content area.

Search options page
- Three browse icons moved to sidebar: The original displays Most Popular, New Releases, and Random Suggestions as navlink icons above the search tabs. The edited version removes them from the top entirely and instead places them in a new right-hand sidebar.
- Sidebar added: The edited version introduces a page-with-sidebar flex layout with a new browse-sidebar panel on the right. Each sidebar item (Most Popular, New Releases, Random Suggestions) includes an example book cover and author (Pride and Prejudice / Jane Austen, The Mysterious Affair at Styles / Agatha Christie, Moby Dick / Herman Melville).
- "Back" button added: Same as in the Frankenstein edit — a back button is added at the top of the content area.
