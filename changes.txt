Page specific changes:

Landing page html changes
- Added prominent search bar, since most people who will visit the website will want to look up a specific book
- wrapped the slogan in a info-box container so it fits the theme
- removed redundant info boxes (get help, Audio books, social media). This information was already available in other places and cluttered up the landing page
- simplified the how to help and the special areas info box.

Landing page css changes
-  There is no landing page specific css, it all uses global css.

book page html changes
- QR code removed: the original shows a QR code block; the edited version
  does not.
- Social share icons: the original uses CSS sprite-based icons. The edited
  version replaces all of them with inline icons.
- "Share" heading added: a "Share this book with your friends" label is above the social buttons.
- "Summary" heading added: an <h2 class="summary-heading">Summary</h2> is
  added before the book description. The original has no heading there.
- Download section redesigned: the original uses a full HTML table listing
  all formats. The edited version replaces this with a "Read Now!"   button and a collapsible <details> dropdown for all download options, styled as individual rows instead of table cells.
- "Similar Books" section redesigned: replaced the list with links with a horizontally   scrollable book carousel and a collection of category pill-buttons.
- Subjects table row consolidated: the original lists each subject in a
  separate <tr>. The edited version puts all subjects in a single cell
  separated by bullet (•) characters.
